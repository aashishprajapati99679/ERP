# College ERP

A comprehensive Enterprise Resource Planning (ERP) application designed to streamline college operations. This system provides features such as student registration with email OTP verification, administrative workflows, and a modern, responsive user interface.

## Live Demo
Check out the live application here: **[https://college-erp-601x.onrender.com/](https://college-erp-601x.onrender.com/)**

## Technology Stack

- **Backend:** Python, Django
- **Database:** MongoDB (using Djongo connector)
- **Frontend:** HTML, CSS, JavaScript (Custom Warm Theme)
- **Deployment:** Render

## Features

- **Student Registration:** Secure student registration workflow including Email OTP Verification.
- **Administrative Dashboard:** Manage students, courses, and other college resources efficiently.
- **Modern UI:** A cohesive, warm color palette providing a clean and intuitive user experience.
